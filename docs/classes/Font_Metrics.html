\Font_Metrics

The font metrics class

This class provides information about fonts and text. It can resolve font names into actual installed font files, as well as determine the size of text in a particular font and size.

get_font()

get_font(string $family, string $subtype) : string

Resolves a font family & subtype into an actual font file

Subtype can be one of 'normal', 'bold', 'italic' or 'bold_italic'. If the particular font family has no suitable font file, the default font ({@link DOMPDF_DEFAULT_FONT}) is used. The font file returned is the absolute pathname to the font file on the system.

Parameters

string $family
string $subtype

Returns

string
